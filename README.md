
# Telegram Prediction Bot

Бот отправляет предсказания 31 день подряд.  
Разные пользователи получают разные предсказания в один и тот же день.

## Запуск
- Установите зависимости: `pip install -r requirements.txt`
- Вставьте токен в bot.py и scheduler.py
- Запустите бота: `python bot.py`
- Настройте крон на Render для запуска scheduler.py ежедневно
